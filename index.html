Растрові тайли

Переваги

  • Мінімальне навантаження на процесор користувача
  • Простота розробки софту для відображення

Недоліки

  • Дуже великий обсяг, в разі jpeg-тайлів - замилювання
  • Неможливість динамічної зміни стилів

GeoJson

GeoJSON — відкритий формат призначений для зберігання географічних структур даних, заснований на JSON

Типи даних

  • Point
  • LineString
  • Polygon
  • MultiPoint
  • MultiLineString
  • MultiPolygon
  • Feature
  • FeatureCollection

{
	"type": "FeatureCollection",
	"features": [
		{ "type": "Feature",
			"geometry": {...},
			"properties": {"prop0": "value0"}
		},
	]
}

						

{ "type": "Point",
    "coordinates": [30, 10]
}


{ "type": "LineString",
    "coordinates": [
        [30, 10], [10, 30], [40, 40]
    ]
}


{ "type": "Polygon",
    "coordinates": [
        [[30, 10], [40, 40], [20, 40], [10, 20], [30, 10]]
    ]
}

						

{ "type": "MultiPoint",
    "coordinates": [
        [10, 40], [40, 30], [20, 20], [30, 10]
    ]
}


{ "type": "MultiLineString",
    "coordinates": [
        [[10, 10], [20, 20], [10, 40]],
        [[40, 40], [30, 30], [40, 20], [30, 10]]
    ]
}

						

{ "type": "MultiPolygon",
    "coordinates": [
        [
            [[40, 40], [20, 45], [45, 30], [40, 40]]
        ],
        [
            [[20, 35], [10, 30], [10, 10], [30, 5]
            [[30, 20], [20, 15], [20, 25]]
        ]
    ]
}
